Работа с пользователем SlavaBot

/rest/user/v1/send-password/ post

Отправляем на почту пользователю код интеграции

post

post /rest/user/v1/send-password/

  • Request
  • Response

Body

Media type: application/x-www-form-urlencoded

Type: object

Properties
  • login: required (string)

    Логин пользователя SalesBot

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
    "status": true
}

/rest/user/v1/auth-telegram/ post

Привязка telegram ID к пользователю, по паре логин + код интеграции, сброс кода интеграции.

post

post /rest/user/v1/auth-telegram/

  • Request
  • Response

Body

Media type: application/x-www-form-urlencoded

Type: object

Properties
  • login: required (string)

    Логин пользователя SalesBot

  • code: required (string)

    Код интеграции

  • tid: required (string)

    Telegram ID, к которому будет осуществляться привязка

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
    "status": true
}

/rest/user/v1/set-time-zone/ post

Установить пользователю часовой пояс

post

post /rest/user/v1/set-time-zone/

  • Request
  • Response

Body

Media type: application/x-www-form-urlencoded

Type: object

Properties
  • tid: required (integer)

    Telegram ID пользователя

  • timezone: required (string)

    Часовой пояс пользователя в формате Europe/Moscow

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
    "status": false,
    "error": "Timezone error!"
},

{
    "status": false,
    "error": "Telegram ID error!"
},

{
    "status": false,
    "error": "User not found!"
},

{
    "status": true,
    "old_timezone": "test/test",
    "new_timezone": "Europe/Moscow"
}

/rest/accounts/v1/set-account-status/ post

Изменение статуса аккаунта пользователя

post

post /rest/accounts/v1/set-account-status/

  • Request
  • Response

Body

Media type: application/x-www-form-urlencoded

Type: object

Properties
  • wall_id: required (string)

    ID стены пользователя, к которой относится статус

  • status: required (one of 0, 1)

    null Нет данных о доступе 0 Ошибка доступа 1 Доступ активен

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
  "status": true,
}

Работа с историей действий

/rest/history/v1/new-event/ post

Добавление события нового поста в соц сеть.

post

post /rest/history/v1/new-event/

  • Request
  • Response

Body

Media type: application/x-www-form-urlencoded

Type: object

Properties
  • tid: required (integer)

    ID пользователя telegram.

  • type: required (one of facebook, vkontakte, instagram)

    Тип поста соц. сети

  • data: required (string)

    Данные поста, в формате JSON

HTTP status code 200

Body

Media type: application/json

Type: object

Example:

{
    "status": true
},

{
    "status": false,
    "error": "Data not found in request!"
},

{
    "status": false,
    "error": "Data is not JSON!"
},

{
    "status": false,
    "error": "Type not found in request!"
},

{
    "status": false,
    "error": "Type error! It must be equals facebook, vkontakte or instagram!"
},

{
    "status": false,
    "error": "Telegram ID not found in request!"
},

{
    "status": false,
    "error": "User not found in database!"
}
